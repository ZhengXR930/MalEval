MalEval Artifact, version 1.0.0
================================

This is the reproduction artifact for the ISSTA 2026 article:

  Is “Knowing It’s Malicious” Enough? Evaluating LLMs for Fine-Grained
  Malware Behavior Auditing

Article DOI: https://doi.org/10.1145/3832187

The source code, sample metadata, prompts, and evaluation scripts are in this
repository. The associated large dataset is hosted at:

  https://huggingface.co/datasets/Xinzxr/MalEval

Use the camera-ready dataset revision:

  a5bd8d81116d2936a3edcd7eae5b26403aabdbd4

Complete installation, execution, verification, citation, and safety
instructions are in README.md, DATA.md, LICENSE.txt, and SECURITY.md in the
root of this package.

IMPORTANT: This artifact concerns Android malware. Do not install or execute
APK samples on personal or production devices. Use an isolated research
environment and follow all institutional and legal requirements.
